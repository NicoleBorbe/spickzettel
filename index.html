Spickzettel für die wichtigsten Terminal-Befehle

Befehl Erklärung
ls Inhalt eines Ordners anzeigen
ls -1 Inhalt eines Ordners in einer Spalte anzeigen
cd Ordner wechseln
cd .. In der Ordnerstruktur eine Stufe zurück
cd ~ Zurück zum User-Ordner
mkdir Einen Ordner erstellen
touch Eine Datei erstellen
mv Eine Datei verschieben oder umbenennen
cp Eine Datei oder Ordner kopieren
rm Eine Datei löschen
rmdir Einen Ordner löschen
clear Bildschirm des Terminals leeren
pwd Wo bin ich?
